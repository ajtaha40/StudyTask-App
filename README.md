# Study Task App

A full-featured web-based task management application designed to help students organize, track, and manage their study tasks efficiently.

---

## 🚀 Project Versions

### ✅ Version 1 (Core Features)

* Add and delete tasks
* Mark tasks as completed
* Basic task list interface

---

### ✅ Version 2 (Enhanced Functionality)

* Due date assignment for tasks
* Priority levels (Low, Medium, High)
* Search functionality
* Task filtering

---

### ✅ Version 3 (Final Version)

* Improved UI/UX design
* Dark mode toggle
* Dashboard summary (task insights)
* Simple login / user profile system
* Automated testing (Jest)
* Continuous Integration using GitHub Actions

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Node.js (for package management)
* Jest (for testing)
* GitHub Actions (CI/CD)

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/ajtaha40/StudyTask-App.git
```

2. Navigate into the project folder:

```
cd study-task-app
```

3. Open `index.html` in your browser

---

## 🧪 Running Tests

Install dependencies and run tests:

```
npm install
npm test
npm start
```

---
